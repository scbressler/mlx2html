- Two sliders: Square Wave Frequency (1–6 Hz) and Number of Harmonics (1–30)
- A dropdown: Time Domain / Frequency Domain
- A "Plot It" button
- Time domain plot: individual harmonics in gray + summed square wave in blue
- Frequency domain plot: FFT magnitude stem plot
- Fourier series math: y(k) = (4/π)(1/(2k-1)) sin(2π(2k-1)ft)

Built and delivered squareWaveBuilder.html using Chart.js, with:
- Dark theme UI (Space Mono + DM Sans fonts)
- Frequency slider: 1–6 Hz
- Harmonics slider: 1–30
- Time/Frequency domain toggle
- "Plot It →" button
- FFT implemented via Cooley-Tukey radix-2 algorithm (replacing MATLAB's fft())
- Gibbs phenomenon annotation in time domain view
